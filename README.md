Partie 1 — Sans "Problem Dampener"

Un rapport (une ligne de nombres) est sûr si :

Les valeurs sont strictement croissantes ou strictement décroissantes.

Les différences entre nombres consécutifs sont comprises entre 1 et 3 inclus.

Aucune paire de valeurs consécutives n’est égale.

Partie 2 — Avec "Problem Dampener"

Un rapport est considéré sûr si :

Il respecte les règles ci-dessus ou

Il devient sûr après suppression d’un seul nombre (n’importe lequel de la ligne).
